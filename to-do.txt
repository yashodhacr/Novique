4. PWA ICONS (for Android "Add to Home Screen" and install prompts)
--------------------------------------------------------------------
Files:
  frontend/public/icon-192.png  (192x192 px)
  frontend/public/icon-512.png  (512x512 px)
After adding these, update frontend/public/manifest.json:
  Replace the "favicon.ico" entry with:
  {
    "src": "/icon-192.png",
    "sizes": "192x192",
    "type": "image/png",
    "purpose": "any maskable"
  },
  {
    "src": "/icon-512.png",
    "sizes": "512x512",
    "type": "image/png",
    "purpose": "any maskable"
  }
Notes:
  - Use "maskable" purpose so Android can apply adaptive icon shapes
  - Keep logo within the center 80% circle ("safe zone") for maskable icons
  - Tool: https://maskable.app to test maskable icon appearance
